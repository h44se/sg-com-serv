# Formula One Dashboard Workspace

This workspace contains the implementation plan and the first scaffold for the Formula One dashboard.

## Status

- Plan created: `implementation-plan.md`
- Provider validation started: OpenF1 endpoints were probed and the working / failing shapes were noted in the plan.
- Initial implementation scaffold: in progress

## Current focus

1. Backend provider/client contract
2. Frontend timezone and dashboard view-model types
3. Test fixtures for time and provider normalization

## Suggested profile split

- `product-analyst` — scope and source validation
- `architect` — contract and cache policy
- `backend-engineer` — provider adapter and API assembly
- `frontend-engineer` — dashboard UI and timezone logic
- `test-engineer` — transform and UI tests
- `devops-engineer` — runbook and packaging
- `code-reviewer` — final review
